HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content showed to the user.
  • The title element defines the title of the webpage. This element is required and important for SEO.
  • The style element defines the CSS styles associated the a website.
  • The base element defines the base URL for a webppage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The script element embeds client-side scripts in the HTML document.
  • Elements that are nested inside other elements are called child elements.
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
  • The img element contains information about images that are displayed on a webpage.
  • The br element creates a line of empty space, or line break, between two block of content.
  • The a element creates links to the same webpage or other webpages.
  • When using headings be sure to use heading tags in order from h1 to h6 to make the page accessible to users who access the page using screen readers.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The most common way to style a webpage is by using an external CSS style sheet.
  • When using an external style sheet, you link the file using a link element placed in the HTML's head.
  • Every HTML element is essentially a box. Each box consists of margins outside the element, and the element's borders, padding, and content.
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git status: shows modified files in working directory, staged for your next commit.
  • cd(change directory) navigates between directories in the path.
  • cd.. navigates backwards one level at a time through the path of your directories.
  • touch creates a file inside a directory.
  • git clone is used to clone a repository when cloning a repo you need to be inside the directory you want the repo directory to be in.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is read from top to bottom.
  • An if statement is a way of introducing decision-making into our code.
  • An array is a single variable that is used to hold a group of data.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • A function's name is followed by parentheses.
